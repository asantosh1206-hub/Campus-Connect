# Campus Connect AI Pro

A professional Campus Connect AI Hackathon website built with Next.js and Tailwind CSS.

## Pages
- Home
- AI Chat
- Dashboard
- Resume Screening
- Student Portal
- Recruiter Portal

## Deploy on Vercel
1. Upload all files to your GitHub repository.
2. Import the repository in Vercel.
3. Vercel will detect Next.js.
4. Click Deploy.

## IdeaBoxAI Integration
The chatbot is currently demo mode. To connect your real IdeaBoxAI persona, ask organizers for:
- API endpoint
- API key
- Persona ID
- Embed widget code
